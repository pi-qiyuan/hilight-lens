# 🔍 Hilight Lens: Your Visual Intuition Tool

Tired of hunting for key information in long, complex webpages? Hilight Lens is a powerful, lightweight browser extension designed to help you instantly lock onto keywords across any website.

Whether you're scanning massive documents, tracking professional terms, or marking complex names and vocabulary in foreign languages, Hilight Lens builds a visual index in the information chaos, making deep reading and info retrieval easier than ever.

### 💡 CORE LOGIC: Cognitive Unloading
-----------------------------------
When reading difficult articles, most of your brain's bandwidth is wasted on low-level tasks like "**identifying who this is**" or "**what this term means**".

Hilight Lens uses color coding to transform "**logical recognition**" into "**visual intuition**", freeing up your brain's bandwidth to process higher-level logical thinking.

> **Don't let trivial nouns wear down**

### 🌟 CORE USAGE SCENARIOS
-----------------------------------
- ✅ **Professional Work**: Quickly locate key parameters and clauses in technical docs, codebases, or legal terms.
- ✅ **Language Learning**: Lock high-frequency vocabulary when reading foreign literature to eliminate reading barriers.
- ✅ **Deep Reading**: Track complex character relationships and location changes in long novels.
- ✅ **Info Tracking**: Instantally capture specific stock codes, brand names, or price points in massive news feeds.

### 🚀 MODERN, FAST & SUPERIOR EXPERIENCE
-----------------------------------
Built with the latest standards for ultimate performance and stability:

- ✨ **Multiple Keywords**: Support for highlighting unlimited keywords simultaneously with independent colors for each.

- ✨ **One Add, Everywhere**: Add a keyword on any page, and it will sync across all tabs and websites instantly. No repeated work needed.

- ✨ **Instant Response**: Highlights are applied immediately without manual page refreshes when adding or modifying keywords.

- ✨ **Millisecond Auto-Highlight**: Uses advanced algorithms to detect dynamically loaded content without slowing down your browser.

### 🛠️ KEY FEATURES
-----------------------------------
- ✅ **Multi-Keyword Management**: Build a clear visual map with multiple terms and non-interfering colors.
- ✅ **Context Menu**: Select any text and right-click to instantly highlight it across the entire web.
- ✅ **Smart Presets**: Built-in beautiful color schemes with auto-rotation, plus custom background/text colors.
- ✅ **Local Persistence**: Keywords are automatically saved and synced locally, keeping all tabs consistent.
- ✅ **Privacy First**: All processing happens locally. We do NOT collect or upload any user data.
- ✅ **Multilingual**: Interface available in English, 中文, 日本語, 한국어, Deutsch, Français, and Español.

### Installation
[Download on Chrome Web Store](https://chromewebstore.google.com/detail/special-character-picker/ejofkafmbaaipjkegidfegeilldhcacm)

### 📖 QUICK START
-----------------------------------
1. Click the toolbar icon, enter a keyword in the popup to add it quickly.
2. Select text on any webpage, right-click, and choose "Highlight...".
3. Manage, modify colors, or delete keywords directly in the popup panel.
